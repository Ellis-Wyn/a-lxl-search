## 1. 产品概述

### 1.1 产品简介

A_lxl_search 是一个专注于肿瘤药物研发管线的智能检索系统，通过聚合多源数据（靶点、文献、管线、CDE事件），为药物研发人员提供一站式的竞争情报分析平台。

### 1.2 核心价值

- **统一检索**: 一次搜索同时查询管线、文献、靶点、CDE事件
- **智能归一化**: 自动标准化异构数据格式（Phase、适应症、公司名）
- **实时监控**: 增量监控管线变化，自动检测竞品退场
- **高性能**: Redis 缓存加速，响应时间减少 60-80%
- **可追溯**: 所有数据保留原始来源 URL

### 1.3 目标用户

- 药物研发人员
- 竞争情报分析师
- 临床开发团队
- 投资研究机构

### 1.4 技术栈

| 组件 | 技术 | 版本 |
|------|------|------|
| 后端框架 | FastAPI | 0.104+ |
| 数据库 | PostgreSQL | 15+ |
| 缓存 | Redis | 7+ |
| 爬虫 | Playwright / Scrapy | 1.40+ / 2.11+ |
| 任务调度 | APScheduler | 3.10+ |
| ORM | SQLAlchemy | 2.0+ |
| 容器化 | Docker / Docker Compose | Latest |

### 3.2 功能详细说明

#### 3.2.1 统一搜索（Unified Search）

**功能描述**: 一次调用同时搜索管线、文献、靶点、CDE事件

**特性**:
- 智能查询扩展（同义词、全名自动匹配）
- 相关性评分排序
- 多维度筛选（公司、阶段、日期、事件类型）
- 自动缓存加速（第二次查询提速 60-80%）

**使用场景**:
- 快速了解某个靶点的整体研发态势
- 竞争情报分析（某公司的所有管线）
- 监管进展跟踪（IND/NDA/BLA 申请）

#### 3.2.2 数据归一化（Data Normalization）

**功能描述**: 自动将异构数据转换为标准格式

**归一化规则**:
- **Phase**: 临床I期 → Phase 1, II期 → Phase 2
- **适应症**: 非小细胞肺癌 → NSCLC
- **公司**: 恒瑞 → 江苏恒瑞医药股份有限公司

**支持的公司**: 12家（恒瑞、百济神州、信达、君实、康方、再鼎、和黄、亚盛、药明、先声、石药、复星）

#### 3.2.3 爬虫调度系统

**功能描述**: 基于 APScheduler 的定时任务系统

**特性**:
- Cron 表达式配置
- 并发控制（限制最大并发数）
- 失败自动重试（指数退避）
- 执行历史记录

**默认调度**:
- CDE 爬虫: 每天凌晨 2 点
- PubMed 更新: 每 6 小时

#### 3.2.4 CDE 自动爬取

**功能描述**: 使用 Playwright 渲染 JavaScript，自动抓取 CDE 事件

**支持的事件类型**:
- IND (新药临床试验)
- CTA (临床试验申请)
- NDA (新药上市申请)
- BLA (生物制品许可申请)

**特性**:
- 自动解析表格数据
- 提取药物名称、申请人、受理号、承办日期
- 保留原始页面 URL

### 6.2 核心 API 接口

#### 6.2.1 统一搜索 API ⭐

**接口**: `GET /api/search/unified`

**功能**: 一次调用搜索所有实体类型（管线、文献、靶点、CDE事件）

**请求参数**:

| 参数 | 类型 | 必需 | 说明 | 示例 |
|------|------|------|------|------|
| q | string | ✅ | 搜索关键词 | EGFR |
| type | string | ❌ | 实体类型 | all/pipeline/publication/target/cde_event |
| company | string | ❌ | 公司名称筛选（仅管线） | 恒瑞医药 |
| phase | string | ❌ | 阶段筛选（仅管线） | Phase 1/2/3 |
| moa_type | string | ❌ | MoA类型筛选（仅管线） | Small Molecule/ADC |
| journal | string | ❌ | 期刊筛选（仅文献） | Nature Medicine |
| date_from | string | ❌ | 起始日期 YYYY-MM-DD | 2024-01-01 |
| date_to | string | ❌ | 结束日期 YYYY-MM-DD | 2024-12-31 |
| event_type | string | ❌ | CDE事件类型 | IND/CTA/NDA/BLA |
| limit | int | ❌ | 每类结果数量限制 | 20 |

**请求示例**:

```bash
# 搜索 EGFR 相关所有内容
curl "http://localhost:8000/api/search/unified?q=EGFR&type=all&limit=10"

# 只搜索恒瑞医药的管线
curl "http://localhost:8000/api/search/unified?q=EGFR&type=pipeline&company=恒瑞医药"
```

**现象**: `Error connecting to Redis: Error 111 connecting to localhost:6379`

**解决方案**:

```bash
# 检查 Redis 容器状态
docker ps | grep redis

# 如果未运行，启动 Redis
docker-compose up -d redis

docker-compose exec redis redis-cli ping
# 预期输出: PONG

netstat -ano | findstr :6379  # Windows
lsof -i :6379                 # Linux/Mac
```
